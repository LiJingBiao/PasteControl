# PasteControl
一款轻量级的Mac端剪贴板管理应用

### 当前版本0.1.0，可在 [releases](https://github.com/ZJM6658/PasteControl/releases)中下载试用

如图

- [x] 使用CoreData进行数据存取
- [x] 支持自定义屏幕四个方向的显示方式
- [x] 支持自定义唤起热键
- [x] 支持Delete键删除选中项
- [x] 支持一键清理所以记录
- [x] 列表随滚轮滚动
- [x] 前向（上、左），后向（下、右）方向键切换选中项
- [x] Enter键和Return键复制选中条目内容
- [x] 双击复制条目内容
- [x] 复制成功发送通知
- [x] 支持一键搜索选中项（默认Google）
- [ ] 支持选择搜索引擎
- [ ] 容量控制
- [ ] 主色自定义
- [ ] Undo/Redo 支持
- [ ] 记录剪贴板中的图片
